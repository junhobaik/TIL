## Major Rule
- Markdown 문서로 파일 작성
- `ROOT/기술폴더/문서`의 구조로 배치
- ROOT Directory 문서 배치 하지 않는다
- 기술 폴더명, 문서 파일명 모두 **영문, 소문자**

## Commit Rule
- 새로운 TIL 추가
    - `Add [javascript] scope`
- 기존 TIL에 내용 추가
    - `Update [javascript] scope`
- 기존 TIL의 **잘못된** 내용 수정
    - `Fix [javascript] scope`
